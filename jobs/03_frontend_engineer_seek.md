## Job Title
Front-End Engineer - React (Full-time)

## About the Role
Creative Digital Agency partners with major Australian and international brands to deliver cutting-edge digital experiences. We're looking for an experienced Front-End Engineer to lead the development of responsive, high-performance web applications for our diverse clientele. You'll be part of a collaborative team of designers, developers, and strategists.

## Key Responsibilities
- Architect and implement scalable React components and applications
- Collaborate with UX/UI designers to translate designs into interactive experiences
- Optimize web applications for performance, accessibility, and SEO
- Manage state management using Redux, Context API, or Zustand
- Write comprehensive unit and integration tests
- Participate in design reviews and technical planning sessions
- Ensure code quality through peer reviews and adherence to coding standards
- Troubleshoot and debug browser compatibility issues
- Stay updated on latest React ecosystem trends and technologies

## Required Skills
- Expert-level proficiency in React and JavaScript/TypeScript
- Deep understanding of HTML5, CSS3, and modern CSS frameworks (Tailwind, Styled Components)
- Strong experience with state management libraries (Redux, Context API, Recoil)
- Familiarity with testing libraries (Jest, React Testing Library, Cypress)
- Understanding of responsive design and mobile-first approaches
- Experience with REST APIs and GraphQL
- Knowledge of web performance optimization techniques
- Proficient with Git and GitHub workflows
- Understanding of build tools (Webpack, Vite, or similar)
- Knowledge of accessibility (WCAG) standards

## Experience & Qualifications
- 4+ years of professional front-end development experience
- Proven expertise in React (2+ years in production)
- Bachelor's degree in Computer Science or related field (or equivalent experience)
- Portfolio or GitHub profile demonstrating React expertise
- Experience with Next.js or Gatsby is highly valued
- Familiarity with TypeScript and static typing
- Experience with design systems and component libraries
- Knowledge of agile development practices
