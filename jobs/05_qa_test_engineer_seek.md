## Job Title
QA Test Engineer (Full-time)

## About the Role
Quality Software Solutions is an independent software testing and quality assurance firm serving financial services, healthcare, and e-commerce sectors. We're recruiting a QA Test Engineer to design and execute comprehensive test strategies for complex enterprise applications. You'll ensure our clients' software meets the highest standards of reliability, security, and user experience.

## Key Responsibilities
- Develop and execute comprehensive test plans and test cases for web and mobile applications
- Perform manual functional, regression, integration, and performance testing
- Identify, document, and report defects using test management tools (JIRA, TestRail)
- Collaborate with developers and product managers to clarify requirements
- Automate test cases using Selenium, Cypress, or TestNG
- Participate in API testing using Postman or SoapUI
- Execute load and performance testing using JMeter or LoadRunner
- Participate in UAT (User Acceptance Testing) coordination
- Maintain test documentation and test case repositories
- Contribute to continuous improvement of QA processes and methodologies

## Required Skills
- Manual and automated software testing experience
- Proficiency in test automation frameworks (Selenium WebDriver, Cypress, TestNG)
- API testing tools (Postman, REST Assured, SoapUI)
- Test management tools (JIRA, TestRail, Azure Test Plans)
- SQL and database testing knowledge
- Basic understanding of software development lifecycle (SDLC)
- Experience with Agile/Scrum methodologies
- Strong analytical and problem-solving skills
- Attention to detail and excellent communication abilities
- Familiarity with CI/CD pipelines and automated testing integration

## Experience & Qualifications
- 2-4 years of QA testing experience in software development
- Strong background in manual and automated testing
- Experience testing web applications and preferably mobile apps
- Knowledge of various testing types (functional, regression, performance, security)
- ISTQB certification or equivalent testing knowledge
- Experience with at least one programming language (Java, Python, JavaScript)
- Bachelor's degree in Computer Science, Information Technology, or related field
- Familiarity with test metrics and KPIs
- Experience in financial services or highly regulated industry is a plus
