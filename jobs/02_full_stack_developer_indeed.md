## Job Title
Full Stack Developer (Mid-Level)

## About the Role
DataFlow Systems is a fintech company focused on data analytics and business intelligence. We're looking for a skilled Full Stack Developer to build robust web applications and services that power our analytics platform. You'll work with modern technologies and have the opportunity to impact how businesses make data-driven decisions.

## Key Responsibilities
- Design and implement scalable backend services using Node.js or Python
- Develop responsive front-end interfaces using React or similar frameworks
- Collaborate with product managers and data engineers on feature specifications
- Optimize application performance and database queries
- Implement automated testing and continuous integration/deployment pipelines
- Mentor junior developers and conduct code reviews
- Troubleshoot and resolve production issues
- Document technical solutions and architecture decisions

## Required Skills
- Strong proficiency in JavaScript/TypeScript or Python
- Experience with React, Vue.js, or similar front-end framework
- Backend development experience (Node.js, Django, Flask, or equivalent)
- Relational database design (PostgreSQL, MySQL)
- Experience with REST API design and development
- Solid understanding of SQL and database optimization
- Familiarity with containerization (Docker) and orchestration (Kubernetes is a plus)
- Experience with version control (Git) and CI/CD tools
- Understanding of software design patterns and clean code principles
- Problem-solving and analytical mindset

## Experience & Qualifications
- 3-5 years of professional software development experience
- Proven track record of delivering production applications
- Bachelor's degree in Computer Science, Engineering, or related field (or equivalent experience)
- Experience with AWS, GCP, or Azure cloud platforms
- Knowledge of agile methodologies
- Experience with testing frameworks (Jest, pytest, etc.)
- Familiarity with message queues (RabbitMQ, Kafka) is a plus
