# DevOps Engineer

## Company
Cloud Infrastructure Inc.

## Job Source
**Platform:** LinkedIn  
**Posted:** February 2026  
**Location:** Perth, WA, Australia

## Job Title
DevOps Engineer (Full-time)

## About the Role
Cloud Infrastructure Inc. helps enterprises migrate and manage their applications on cloud platforms. We're seeking a skilled DevOps Engineer to design, implement, and maintain our infrastructure and deployment pipelines. You'll work with cutting-edge cloud technologies and help our clients achieve operational excellence.

## Key Responsibilities
- Design and maintain CI/CD pipelines using Jenkins, GitLab CI, or GitHub Actions
- Manage cloud infrastructure on AWS, Azure, or GCP (Infrastructure as Code)
- Implement and manage containerization using Docker and Kubernetes
- Monitor system performance, logging, and alerting using ELK, Prometheus, Grafana
- Automate infrastructure provisioning and deployment processes
- Collaborate with development teams to optimize deployment processes
- Manage security and compliance requirements for cloud applications
- Troubleshoot and resolve production incidents
- Implement disaster recovery and backup strategies
- Participate in on-call rotation for critical infrastructure

## Required Skills
- Strong experience with cloud platforms (AWS preferred, but Azure/GCP acceptable)
- Proficiency in Infrastructure as Code tools (Terraform, CloudFormation, Ansible)
- Expert-level Docker and Kubernetes experience
- Shell scripting (Bash) and at least one programming language (Python, Go, Java)
- Experience with CI/CD tools and practices (Jenkins, GitLab, GitHub Actions)
- Linux system administration and networking knowledge
- Database administration experience (MySQL, PostgreSQL, NoSQL)
- Monitoring and logging tools (Prometheus, ELK, Grafana, CloudWatch)
- Strong problem-solving and troubleshooting skills
- Excellent documentation and communication skills

## Experience & Qualifications
- 4+ years of DevOps/Infrastructure engineering experience
- 3+ years of hands-on cloud platform experience
- Bachelor's degree in Computer Science, Engineering, or related field
- AWS Certified Solutions Architect or equivalent certification
- Experience with multiple programming languages
- Familiarity with security best practices and compliance (AWS Security, ISO 27001)
- Experience with microservices architecture and distributed systems
- Knowledge of GitOps practices is a plus
